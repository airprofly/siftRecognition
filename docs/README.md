# 📖 文档目录

本目录包含场景识别项目的实验报告（LaTeX）和相关图表资源。

## 内容

| 文件 | 说明 |
|------|------|
| `report.tex` | 计算机视觉与模式识别课程实验报告，基于 UESTC 实验报告模板 |
| `uestc_expr_report.cls` | 电子科技大学实验报告 LaTeX 模板 |
| `figures/` | 报告引用的图表资源 |

## 编译报告

```bash
cd docs
xelatex report.tex
```

## 相关项目

该项目为 [siftRecognition](https://github.com/airprofly/siftRecognition) 的文档子模块，核心训练代码位于项目根目录的 `src/` 中。
